Rule new_expression_parentheses¶

All new expressions with a further call must (not) be wrapped in parentheses.

Warning¶

This rule is CONFIGURABLE¶

You can configure this rule using the following option: use_parentheses.

Configuration¶

use_parentheses¶

Whether new expressions with a further call should be wrapped in parentheses or not.

Allowed types: bool

Default value: false

Examples¶

Example #1¶

Default configuration.

--- Original
+++ New
 <?php

-(new Foo())->bar();
+new Foo()->bar();

Example #2¶

Default configuration.

--- Original
+++ New
 <?php

-(new class {})->bar();
+new class {}->bar();

Example #3¶

With configuration: ['use_parentheses' => true].

--- Original
+++ New
 <?php

-new Foo()->bar();
+(new Foo())->bar();

Example #4¶

With configuration: ['use_parentheses' => true].

--- Original
+++ New
 <?php

-new class {}->bar();
+(new class {})->bar();
