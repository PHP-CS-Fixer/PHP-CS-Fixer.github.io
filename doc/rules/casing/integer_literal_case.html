Rule integer_literal_case¶

Integer literals must be in correct case.

Examples¶

Example #1¶

--- Original
+++ New
 <?php
-$foo = 0Xff;
-$bar = 0B11111111;
+$foo = 0xFF;
+$bar = 0b11111111;
