Rule no_spaces_inside_parenthesis¶

There MUST NOT be a space after the opening parenthesis. There MUST NOT be a space before the closing parenthesis.

Examples¶

Example #1¶

--- Original
+++ New
 <?php
-if ( $a ) {
-    foo( );
+if ($a) {
+    foo();
 }

Example #2¶

--- Original
+++ New
 <?php
-function foo( $bar, $baz )
+function foo($bar, $baz)
 {
 }
