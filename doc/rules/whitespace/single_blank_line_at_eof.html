Rule single_blank_line_at_eof¶

A PHP file without end tag must always end with a single empty line feed.

Examples¶

Example #1¶

--- Original
+++ New
 <?php
-$a = 1;
\ No newline at end of file
+$a = 1;

Example #2¶

--- Original
+++ New
 <?php
 $a = 1;
-
