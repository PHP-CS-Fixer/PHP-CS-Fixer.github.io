Rule multiline_string_to_heredoc¶

Convert multiline string to heredoc or nowdoc.

Examples¶

Example #1¶

--- Original
+++ New
 <?php
-$a = 'line1
-line2';
+$a = <<<'EOD'
+line1
+line2
+EOD;

Example #2¶

--- Original
+++ New
 <?php
-$a = "line1
-{$obj->getName()}";
+$a = <<<EOD
+line1
+{$obj->getName()}
+EOD;
