Rule pow_to_exponentiation¶

Converts pow to the ** operator.

Warning¶

Using this rule is risky¶

Risky when the function pow is overridden.

Examples¶

Example #1¶

--- Original
+++ New
 <?php
- pow($a, 1);
+ $a** 1;
